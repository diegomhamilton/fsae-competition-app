# Phase 2B Task Initiation — Evidence Step Promotion

This task is intentionally split from Phase 2A measurement migration so both tracks can proceed in parallel as separate PRs.

## Scope
- Promote candidate `check`/`action` steps to `evidence` where proof artifacts are required.
- First-release evidence classes: `photo`, `video`, `audio`, `pdf`, `text`.

## Deliverables
1. Seed updates converting selected steps to `evidence`.
2. Conversion summary (by seed file, before/after counts).
3. Reviewer sign-off checklist for false-positive prevention.

## Out of scope
- UI capture widgets and persistence model changes (Phase 3).
